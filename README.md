"Uyga vazifa: Git va GitHub asoslari

1. Repozitariy va majburiyatlarni yaratish:
    - GitHub-da yangi ommaviy ombor yarating. Uni ""MyFirstRepository"" deb nomlang.
    - "git init" buyrug'i yordamida kompyuteringizdagi bo'sh papkada mahalliy Git omborini ishga tushiring.
    - `README.md` yangi matn faylini yarating.
    - `README.md` ga ba`zi o`zgartirishlar kiriting (masalan, loyihaning qisqacha tavsifini yozing).
    - Mahalliy omboringizga o'zgartirishlar kiritish va o'zgartirishlar kiritish uchun Git buyruqlaridan foydalaning (`git add .` va `git commit -m ""First commit""`).

2. GitHub-ga majburiyatlarni yuborish:
    - “git remote add origin ” buyrug‘i yordamida mahalliy omboringizni uzoq GitHub omboringizga bog‘lang.
    - “git push -u origin main” buyrug'i yordamida mahalliy ombordan GitHub-ga majburiyatlarni surish.
"
